Translator for a Linear Algebra DSL to a number of output formats.

This project uses ANTLR and StringTemplate to parse the input and produce the output target language.

The Linear Algebra DSL is similar to the matrix manipulation subset of the MATLAB® language.

See mparser by David Wingate for a more complete MATLAB grammar.

Python
CommonsMath

Example direct use: 

      curl http://linalgtrans.appspot.com/Translate -d input="A=rand(10) b=rand(10,1) x=A\b"
